# RunFungusFlowchartItemId

用来跳转原版剧情指定位置用.详细使用建议看看next导出文件`Fungus`文件夹.如果只是跳转Talk的话建议使用[RunFungusTalkItemId](./RunFungusTalkItemId)指令.

## 实例:

<Tabs>
<TabItem value="lua" label="Lua" default>

```lua
function (runner,env)
-- runner.RunFungusFlowchartItemId(flowchartName,tagBlock,Itemid)
-- runner.RunFungusFlowchartItemId(flowchartName,Block名称,Itemid)
-- 指定Itemid开始
runner.RunFungusFlowchartItemId("Talk501","出手相助",322)
end
```

## 参数
| 参数            | 类型      | 描述                     | 选项  |
|---------------|---------|------------------------|-----|
| flowchartName | String  | flowchart名称            | 必填  |
| BlockName     | String  | Block名称                | 必填  |
| ItemID        | Integer | Block里面Command数组ItemId | 必填  |
