# RunFungusItemId
## 描述:
本指令基于RunFungusBlock修改可以指定里面ItemId,用来跳转剧情指定位置用.
## 实例:
<Tabs>

  <TabItem value="lua" label="Lua" default>

```lua
function (runner,env)
  -- runner.RunFungusItemId(tagBlock,Itemid)
  -- runner.RunFungusItemId(Block名称,Itemid)
  --指定Itemid开始
  runner.RunFungusItemId("Talk500","出手相助",322)
end
```

## 参数
| 参数      | 类型    | 描述                       | 选项 |
| --------- | ------- | -------------------------- | ---- |
| BlockName | String  | Block名称                  | 必填 |
| ItemID    | Integer | Block里面Command数组ItemId | 必填 |
