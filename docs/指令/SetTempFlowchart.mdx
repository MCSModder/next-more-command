# SetTempFlowchart
保存当前的Flowchart状态指令,如果需要运行用[RunTempFlowchart](./RunTempFlowchart)指令
## 实例:

<Tabs>

  <TabItem value="lua" label="Lua" default>

```lua
function (runner,env)
  runner.SetTempFlowchart("Talk500")
end
```
  </TabItem>

  <TabItem value="json" label="Json">

```json
[
  {
    "id": "测试1",
    "character": {
      "旁白": 0,
      "主角": 1,
      "倪旭欣": 609
    },
    "dialog": [
      "SetTempFlowchart*Talk500",
    ],
    "option": []
  }
]

```

  </TabItem>
</Tabs>

## 参数
| 参数      | 类型    | 描述                       | 选项 |
| --------- | ------- | -------------------------- | ---- |
| Key       | String  | 键值                       | 必填 |
