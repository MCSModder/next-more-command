# SetNpcTeleport
## 描述:
主要传送单个或者多个NPCID传送指定场景
## 实例:
<!-- tabs:start -->
<!-- tab:Lua -->

```lua
function (runner,env)
  --单个npc
  runner.SetNpcTeleport("S2","112")
  --多个npc 用,来分割多个npcid
  runner.SetNpcTeleport("S2","112,113,114")
end
```

## 参数
| 参数      | 类型   | 描述     | 选项 |
| :--------- | ------ | -------- | -----------: |
| sceneName | String | 场景名字 | 必填 |
| npcId     | String | 填npcId数字,用","分隔符来区分 "112,113,114"         | 必填     |
