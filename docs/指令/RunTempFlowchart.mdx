# RunTempFlowchart
## 描述:
读取保存的[SetTempFlowchart](Command/SetTempFlowchart)指令,运行完就会自动销毁,只能一次性.
## 实例:
<!-- tabs:start -->
<!-- tab:Lua -->

```lua
function (runner,env)
  -- runner.RunTempFlowchart(key,tagBlock,Itemid(可选))
  -- runner.RunTempFlowchart(键值,Block名称,Itemid(可选))
  --默认从第一个Command数组开始
  runner.RunTempFlowchart("Talk500","出手相助")
  --指定Itemid开始
  runner.RunTempFlowchart("Talk500","出手相助",322)
end
```

## 参数
| 参数      | 类型    | 描述                       | 选项 |
| --------- | ------- | -------------------------- | ---- |
| Key       | String  | 键值                       | 必填 |
| BlockName | String  | Block名称                  | 必填 |
| ItemID    | Integer | Block里面Command数组ItemId | 可选 |
