# GetUseSkillID
判断是不是该使用神通ID,这个只能配合技能触发器类型才能用.如果不是会返回false值;
## 实例:
<Tabs>
  <TabItem value="lua" label="Lua" default>

```lua
function (runner,env)
  -- env:GetUseSkillID(skillID)
   env:GetUseSkillID(401)
end
```

  </TabItem>
  <TabItem value="json" label="Json触发器">

```json
[
{
    "id":"测试触发器",
    "type":"玩家技能使用前",
    "condition":" GetUseSkillID(401) ",
    "triggerEvent":"测试1"
}
]
```

  </TabItem>
</Tabs>




## 参数
| 参数      | 类型       | 描述         | 选项  |
|---------|----------|------------|-----|
| skillID | Interger | 神通ID       |     |
| 返回值     | Boolean  | 返回是不是使用该神通 |     |
